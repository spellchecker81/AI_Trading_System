# AI Trading System

## Goal

A desktop application that:

- Connects to Interactive Brokers (IBKR)
- Uses natural language trading rules
- Supports RSI, MACD, Minervini, CAN SLIM, and IBD-style strategies
- Backtests strategies
- Executes trades on an IBKR Paper account
- Can later be switched to live trading after validation

## Development Philosophy

- Test-driven development
- Modular architecture
- Automated testing
- Risk-first design
- Paper trading before live trading
